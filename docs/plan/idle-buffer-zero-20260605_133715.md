# P2-3: IdleBufferFactor=0 导致死锁

**时间**: 2026-06-05 21:37 UTC
**状态**: ✅ 已修复
**文件**: model.go

## 问题描述

```go
bufferSize := int(float64(config.MaxSize) * config.IdleBufferFactor)
```

如果 `IdleBufferFactor=0`，`bufferSize=0`，创建无缓冲 channel。`preInit` 中 `p.resources <- &resource{...}` 会永久阻塞（没有接收者），goroutine 泄漏，池子无法初始化。

## 修复方案

添加下限保护：

```go
bufferSize := int(float64(config.MaxSize) * config.IdleBufferFactor)
if bufferSize < 1 {
    bufferSize = 1
}
```

或者使用更安全的计算方式（至少 1，至多 MaxSize）：
```go
bufferSize := max(1, int(float64(config.MaxSize)*config.IdleBufferFactor))
if bufferSize > int(config.MaxSize) {
    bufferSize = int(config.MaxSize)
}
```

## 影响范围

- model.go: NewPool() 中的 bufferSize 计算

- 无冲突：纯保护性修复
